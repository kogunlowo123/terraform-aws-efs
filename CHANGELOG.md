# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-07

### Added

- Initial release of the terraform-aws-efs module.
- EFS file system with configurable performance and throughput modes.
- Encryption at rest enabled by default with optional KMS key support.
- Mount targets across multiple subnets.
- Access points with POSIX user and root directory configuration.
- Automatic backup policy (enabled by default).
- Cross-region replication support.
- Dedicated NFS security group with VPC CIDR ingress rule.
- Basic, advanced, and complete usage examples.
